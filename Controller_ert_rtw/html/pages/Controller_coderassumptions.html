C 言語標準

ゼロ初期化コードはモデル 'SILBlock' 用に最適化されています。

整数のグローバル変数の初期値は 0 です。 True
動的に割り当てられる int 配列の各要素の初期値は 0 です。 True

仮定が正しくない場合は、[コンフィギュレーション パラメーター] > [コード生成] > [最適化] の設定を使用して、最適化を解除してください。あるいは、仮定が成り立つようにターゲット環境を構成してください。

浮動小数点数

コード生成は、浮動小数点数をサポートするように構成されています。ターゲット環境では、非正規数の計算が行われる可能性があります。

計算された非正規値をゼロにフラッシュします (FTZ)。 False
入力の非正規値をゼロにフラッシュします (DAZ)。 False

仮定が正しくない場合、非正規数が原因でモデルと生成コードのシミュレーション結果が一致しないことがあります。詳細については、「非正規数のパフォーマンス」を参照してください。
